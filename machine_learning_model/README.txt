Find in this folder all the files required to develop the
Machine Learning Model. It is focused on determine possible
heart diseases.

Files' Description:

   ° ML_HeartDiseasePredictor_GitHub.ipynb = Machine Learning notebook
   ° data.csv = Heart diseases processed data
   ° trained_model.sav = Machine Learning Model in binary file


Note: To run the jupyter notebook you need to upload the data.csv
