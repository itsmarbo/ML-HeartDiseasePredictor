# ML-HeartDiseasePredictor
Predictor of heart disease in patients

Heart disease, is one of the most common illnesses faced by humanity. It is estimated that 244.1 million people are living with ischemic heart disease, this can happen anytime, anywhere to anyone. 

This ML model provides the probability of heart disease, by answering brief questions. The ML model was embedded in a webapp, in order to allow patients a timely detection and reduce risks.



All README files provide a brief description of each folder/file.
